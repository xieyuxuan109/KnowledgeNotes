### 内网穿透
#### 1. 下载并解压（以64位系统为例）
```
wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.tgz
tar zxvf ngrok-v3-stable-linux-amd64.tgz
sudo mv ngrok /usr/local/bin/ # 放入系统路径，方便直接调用
```
#### 2. 访问官网
```
https://dashboard.ngrok.com/login
```
```
#配置你的专属令牌 (Authtoken)
#登录 ngrok.com 后台，在"Your Authtoken"页面复制你的令牌
ngrok config add-authtoken <你的令牌>
```
#### 启动隧道
```
ngrok http 8080//端口为系统提供服务端口
```
#### 暴露 SSH 或数据库服务：以 SSH 默认的 22 端口为例，需要使用 TCP 隧道
```
ngrok tcp 22
//此时会得到一个类似 0.tcp.ngrok.io:12345 的地址，你可以通过 ssh 用户名@0.tcp.ngrok.io -p 12345 来远程连接 
```
